Game Reward View Decorator Script Documentation
The Game Reward View Decorator script is a Unity tool that allows non-programmers to dynamically apply decorators to generic rewards in a game. This script provides a sandbox for modifying the appearance of rewards based on various conditions.

Purpose
The purpose of the Game Reward View Decorator script is to apply decorators to a game reward entity view based on certain reward filters. These filters are customizable and allow the user to control which rewards are decorated.

Inputs and Outputs
The inputs for this script include:

m_FiltersBehaviour: an enum that represents how the reward filters should be applied.
m_Filters: a list of IRewardFilter objects.
m_Decorators: a list of IGameRewardViewDecorator objects.
m_ShowLogs: a boolean that determines whether to display log messages.
The output for this script is a decorated game reward entity view.

Filters
The m_Filters list contains objects that implement the IRewardFilter interface. These filters are used to determine whether a reward should be decorated. The IsRequiredReward method checks each reward filter based on the current FiltersBehaviour.

The FiltersBehaviour enum has three options:

NONE: All filters must return false for the reward to be considered for decoration.
ANY: At least one filter must return true for the reward to be considered for decoration.
ALL: All filters must return true for the reward to be considered for decoration.
Decorators
The m_Decorators list contains objects that implement the IGameRewardViewDecorator interface. These decorators are applied to the game reward entity view if the reward passes the filters.

To customize the decorators, you can create new classes that implement the IGameRewardViewDecorator interface and add them to the m_Decorators list in the inspector. You can also modify existing decorator classes to change their behavior.

Resetting Decorators
To reset the decorators to their default state, you can call the Reset() method. This method loops through the m_Decorators list and calls the Reset() method on each object.

With these settings, a reward entity view will be decorated if the player has completed at least one of the specified levels or tasks. The decorators will change the color and animation of the reward entity view.

                    m_MovedObject.SetParent(m_TargetParent, false);
                    
                    durationText.DurationText.GetComponent<RectTransform>().anchoredPosition = Vector2.zero;
                }
            }
        }

        public override void Reset()
        {
            if (m_MovedObject != null)
            {
                m_MovedObject.SetParent(m_OriginalParent, false);
            }

            m_MovedObject = null;
            m_OriginalParent = null;
        }

    [Serializable]
    public class DecoratorMaterialOverride : SimpleGameRewardViewDecorator
    {
        [SerializeField]
        private Material m_Material;
        [SerializeField]
        private bool m_AddTextGradientEffect;
        [SerializeField]
        private TextWithGradientEffect.Type m_GradientEffectType = TextWithGradientEffect.Type.Default;
        
        private Dictionary<Graphic, Material> m_Cache = new();

        private Graphic[] m_CachedGraphics;
        
        
        public override void Apply(GameRewardEntityView view)
        {
            m_CachedGraphics = GetTargetGraphics(view);
            
            
            foreach (var graphic in m_CachedGraphics)
            {
                m_Cache.Add(graphic, graphic.material);
            }

            foreach (var targetGraphic in m_CachedGraphics)
            {
                targetGraphic.material = m_Material;
            }
        }

        public override void Reset()
        {
            if (m_CachedGraphics == null)
            {
                return;
            }
            
            foreach (var cachedGraphic in m_CachedGraphics)
            {
                if (m_Cache.TryGetValue(cachedGraphic, out var material))
                {
                    cachedGraphic.material = material;
                } 
            }
            
            m_Cache.Clear();
        }

        private Graphic[] GetTargetGraphics(GameRewardEntityView view)
        {
            return view.GetAllRewardText()
                .Where(text =>text != null)
                .Select(text =>
                {
                    if (m_AddTextGradientEffect)
                    {
                        var textWithGradientEffect = text.gameObject.GetOrAddComponent<TextWithGradientEffect>();
                        textWithGradientEffect.type = m_GradientEffectType;
                    }
                    return text.GetComponent<Graphic>();
                })
                .ToArray();
        }
    }
